# ImpulsoNica 

ImpulsoNica es una plataforma web diseñada para **conectar talento nicaragüense con empresas locales**.  
El objetivo es facilitar la búsqueda de empleo y la publicación de vacantes en un entorno moderno, responsivo y accesible.

---

##  Características principales

- **Página pública de inicio** con buscador de empleos por rubro y ubicación.
- **Registro de candidatos** para crear perfiles y aplicar a vacantes.
- **Registro de empresas** para publicar ofertas laborales.
- **Panel de administración** con:
  - Tarjetas resumen (candidatos, empresas, vacantes, postulaciones).
  - Gráficos interactivos con estadísticas.
  - Listados de candidatos, empresas, vacantes y postulaciones recientes.
- **Diseño responsivo** adaptado a móviles, tablets y escritorio.
- **Menú hamburguesa** en móviles para una navegación clara y moderna.

---

## Tecnologías utilizadas

- **Frontend:** HTML5, CSS3, JavaScript
- **Gráficos:** [Chart.js](https://www.chartjs.org/)
- **Diseño:** Flexbox y CSS Grid para layouts responsivos
- **Servidor de desarrollo:** Live Server (VS Code)
